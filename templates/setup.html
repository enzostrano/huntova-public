Step 1 — Pick a provider

Cloud, local, or custom — your choice.

Run the model on your own machine. No API key, no usage charges, no data leaves your machine. Auto-detect probes localhost for running servers.

Plug in any OpenAI-compatible endpoint — your own LiteLLM gateway, a self-hosted vLLM instance, an enterprise relay, or a niche provider.

Step 2 — Connect

Connect to provider.

Stored in your OS keychain (macOS Keychain / Windows Credential Manager / Linux Secret Service) when available, or in an encrypted file at ~/.config/huntova/secrets.enc otherwise. Huntova never sees the key — the local server reads it directly when you run a hunt.
